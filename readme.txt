Как смотреть результат?



Установить NodeJS и npm.

Распаковать проект в каталог.

В терминале перейти в каталог с проектом и выполнить npm install. 

Выполнить npm start. Подождать, после запуска сервера. В браузере открыть http://localhost:8000/. Для удачной работы адрес не должен быть занят. Тогда можно будет протестировать приложение вручную.

Выполнить npm test для karma тестирования (chrome и firefox должны быть установлены). Наблюдать в терминале или браузере информацию о результате тестирования. Завершить работу karma.

Для end-to-end тестирования сначала нужно запустить сервер, выполнив npm start. Затем выполнить npm protractor. Подождать обновлений. Подождать открытия браузера (chrome должен быть установлен). Наблюдать в браузере процесс автоматического тестирования. Затем посмотреть отчет в терминале.
